Anleitung schliessen

Du möchtest miitmi auch auf deiner Webseite verwenden?

So geht's:

1. Folgendes Script in <head> einfügen:


        
          <script src='https://unpkg.com/miitmi-web-component@1.2.1/dist/miitmi-web-component.js'></>

2. Web Components in <body> einfügen:


      
        <miitmi-web-component invite="EINLADEN" video="VIDEOCHAT STARTEN"></>
